# controle_coleta
geradoer de etiqueta, agendamento de coleta, consulta de coleta, confirmacao de coleta
📦 Sistema de Coleta e Etiquetas
📖 Descrição
Este projeto é uma aplicação web desenvolvida com HTML, CSS e JavaScript, integrada a um banco de dados acessado pelo navegador.
O sistema foi criado para gerenciar processos de coleta, oferecendo funcionalidades como:
- Gerador de Etiquetas: Criação automática de etiquetas para identificação de coletas.
- Agendamento de Coleta: Interface para marcar datas e horários de coleta.
- Consulta de Coleta: Pesquisa e visualização de coletas agendadas.
- Confirmação de Coleta: Registro da confirmação de coleta realizada.

🚀 Tecnologias Utilizadas
- Frontend: HTML5, CSS3, JavaScript (ES6+)
- Banco de Dados: IndexedDB / LocalStorage (armazenamento no navegador)
- Design Responsivo: Flexbox e Grid
- Controle de Estado: JavaScript puro

⚙️ Funcionalidades
- 📑 Gerador de Etiquetas: Cria etiquetas únicas com código identificador.
- 🗓️ Agendamento de Coleta: Permite ao usuário cadastrar data, hora e local.
- 🔍 Consulta de Coleta: Exibe lista de coletas agendadas com filtros.
- ✅ Confirmação de Coleta: Atualiza status da coleta para "confirmada".
